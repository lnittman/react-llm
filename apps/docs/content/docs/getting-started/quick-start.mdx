## the 30-second version

add this to your html:

```html
<script src="https://unpkg.com/react-llm/dist/react-llm.js"></script>
<script>
  ReactLLM.init({
    providers: {
      openrouter: 'sk-or-...' // get free key at openrouter.ai
    }
  });
</script>
```

that's it! react llm is now active on your page.

## step-by-step guide

<Steps>
<Step>
### get an openrouter api key

openrouter gives you access to 100+ ai models with one key:

1. go to [openrouter.ai](https://openrouter.ai)
2. click "sign up" (free)
3. create an api key
4. copy the key (starts with `sk-or-`)

<Callout type="info">
  **why openrouter?** one key for claude, gpt-4, gemini, and more. switch models anytime.
</Callout>
</Step>

### add to your website

choose your setup method:

</Tab>
<Tab value="React App">
in your main app file:

```jsx
import { useEffect } from 'react';

function App() {
  useEffect(() => {
    // dynamically load react llm
    const script = document.createElement('script');
    script.src = 'https://unpkg.com/react-llm/dist/react-llm.js';
    script.onload = () => {
      window.ReactLLM.init({
        providers: {
          openrouter: 'sk-or-YOUR-KEY-HERE'
        }
      });
    };
    document.body.appendChild(script);
  }, []);

  return <YourApp />;
}
```
</Tab>

## common questions

### is my api key secure?

yes - your key stays in your browser and is only used for direct api calls.

### which model should i use?

start with claude 3.5 sonnet for best results. it's fast and great at code.

### can i use multiple providers?

yes! add multiple keys:

```javascript
ReactLLM.init({
  providers: {
    openrouter: 'sk-or-...',
    openai: 'sk-...',
    anthropic: 'sk-ant-...'
  }
});
```

### how do i change models?

click the model selector in the chat interface to switch between available models.

## quick configuration

customize react llm for your needs:

```javascript
ReactLLM.init({
  providers: {
    openrouter: 'sk-or-...'
  },
  
  // appearance
  theme: 'dark', // or 'light', 'auto'
  position: 'bottom-left', // or 'bottom-right'
  
  // behavior
  mode: 'development', // enables file editing
  hotkey: 'cmd+shift+l', // custom keyboard shortcut
  
  // defaults
  defaultModel: 'claude-3-5-sonnet',
  temperature: 0.7
});
```
