## quick start

get react llm running in under 30 seconds:

<Steps>
<Step>
### add the script

add this single line to any html page:

```html
<script src="https://unpkg.com/react-llm/dist/react-llm.js"></script>
<script>
  ReactLLM.init({
    providers: {
      openrouter: 'sk-or-...' // get your key at openrouter.ai
    }
  });
</script>
```
</Step>

<Step>
### select a component

click the react llm icon in the bottom-right corner, then click any component on your page.
</Step>

<Step>
### start chatting

ask questions about the component, request changes, or explore the codebase structure.
</Step>
</Steps>

## example use cases

### development mode
- **live code editing**: chat to modify components, see changes instantly
- **debugging assistance**: understand component state and props
- **code generation**: create new components and features through conversation
- **refactoring help**: get ai suggestions for better patterns

### production exploration
- **component discovery**: explore how any website is built
- **pattern library**: save interesting component patterns
- **learning tool**: understand react implementations
- **competitive analysis**: see how others solve ui challenges

## how it works

1. **component detection**: uses react fiber tree traversal to find all components
2. **visual selection**: canvas overlay lets you click to select components
3. **context building**: extracts component code, props, state, and relationships
4. **ai conversation**: sends context to your chosen llm for intelligent responses
5. **live updates**: in dev mode, applies changes directly to your files

## supported models

react llm supports all major ai providers through openrouter:

- **anthropic**: claude 3.5 sonnet, claude 3 opus, claude 3 haiku
- **openai**: gpt-4o, gpt-4 turbo, gpt-3.5 turbo
- **google**: gemini 2.0 flash, gemini 1.5 pro, gemini 1.5 flash
- **meta**: llama 3.1 405b, llama 3.1 70b, llama 3.1 8b
- **and 100+ more models** via openrouter's unified api

## browser compatibility

react llm requires modern browser apis available in:

- chrome/edge 120+
- firefox 125+
- safari 18+
- arc, brave, opera (latest versions)

## philosophy

we believe ai coding assistance should be:

1. **accessible**: works anywhere without complex setup
2. **visual**: select what you want to change by clicking
3. **contextual**: ai understands your specific component
4. **immediate**: see results instantly, not in a separate editor
5. **flexible**: use any ai model that works best for you
